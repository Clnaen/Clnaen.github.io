- 3. 검색

  - **검색과 키**	

    ------

    특정한 조건에 기반하여 데이터 콜렉션으로부터 특정 정보를 선택하는 과정

    "key" : 주목하는 항목	

  - **선형 검색 linear search**

    ------

    **무작위로 늘어놓은 데이터 모임**에서 검색을 수행.

    본 방법은 **순차 콜렉션(sequence) 위에서 반복 수행**하며 특정 아이템을 찾을 때 까지 한 번에 하나씩 혹은 모든 아이템을 찾을 때 까지 수행함.
